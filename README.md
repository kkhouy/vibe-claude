# vibe-claude

A small collection of HTML pages built with Claude.

## Pages

### `index.html`
A minimal hello-world page — red text, clean slate.

### `calculator.html`
A sleek dark-themed calculator with a blue glow aesthetic.

**Features:**
- Addition, subtraction, multiplication, division
- Percentage (`%`) and sign toggle (`+/-`)
- Live expression display above the result
- Chained operations
- Keyboard support (`0–9`, `+ - * /`, `Enter`, `Escape`, `Backspace`)
- Responsive font sizing for long numbers

## Usage

Just open either `.html` file in your browser — no build step, no dependencies.
